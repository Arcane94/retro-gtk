# Retro

Retor is a GObject based Libretro wrapper for frontends. 

## Compiling

To configure the compilation, do:
`./autogen.sh`

You can specify the installation prefix by doing:
`./autogen.sh --prefix /my/prefix`

Then compile:
`make`

## Installing

`make install`
